Data scientist / analyst

El análisis de datos tiene como objetivo facilitar la toma de decisiones de una empresa. Su trabajo consiste en implementar sistemas de recopilación de datos y analizar sus resultados para cumplir con un objetivo determinado.
Estudiar análisis de datos, implica dominar dos disciplinas: la programación y las matemáticas. Pese a esto, el nivel de conocimientos de programación requeridos no son tan extensos como en el caso de otras ramas de estudios tecnológicos como el desarrollo de software. Solamente se requiere un nivel básico con el que implementar un sistema de recopilación de datos, tarea que en ciertas ocasiones llevará a cabo un programador propiamente dicho.
La importancia del trabajo de un data scientist se basa en el análisis de datos para tomar decisiones, lo que implica también tener una base de conocimientos de marketing y empresa. Además, deberán poder expresar la información en términos estadísticos y sacar conclusiones, por lo que la base matemática deberá ser extensa.
